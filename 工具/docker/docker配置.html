docker配置

docker配置

  1. 默认配置文件位置

    /usr/lib/systemd/system

    如果不清楚可以使用命令

     systemctl disable docker
 或
 systemctl enable docker

    重构docker配置文件的软连接然后修改

    修改完成后重新启动docker

     systemctl daemon-reload
 systemctl start docker

    但是已经不推荐直接修改配置文件


  2. 使用外置配置文件

    用命令查看(找到 --config-file )

     man dockerd

    一般默认在 (没有就手动创建)

    /etc/docker/daemon.json


  3. daemon.json 配置详情 daemon.json 和使用docker 命令行参数启动并无区别 (json文件不用每次都输入) 官网解释

    To run the daemon with debug output, use dockerd -D or add "debug": true to the daemon.json file.

    如果要启动debug模式输出运行, 请使用 dockerd-d 或者将 "debug": true 加入到daemon.json

    所以要修改什么可以查看下面命令然后添加到daemon.json中 修改完成后也要重新启动docker

     systemctl restart docker.service
